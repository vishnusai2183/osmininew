Paging

Paging is a memory management scheme that allows non-contiguous allocation of physical memory.

  • The Physical Address Space (main memory) is divided into a number of fixed-size blocks, called frames.
  • The Logical address Space (secondary memory) is divided into fixed-size blocks, called pages.
  • Page Size = Frame Size

    • The mapping from logical to physical address is done by the memory management unit (MMU) which is a hardware device.
